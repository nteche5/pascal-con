# Public Assets Directory

This directory contains static assets for the website.

## Required Assets

### Logo Files
- `logo-primary.svg` or `.png` - Primary logo (PKSΛ + Company Name)
- `logo-secondary.svg` or `.png` - Secondary logo (PKSΛ standalone)
- `favicon.ico` - Favicon for browser tabs

### Images
- Project images for the portfolio (add to `projects/` subdirectory)
- Hero background images (optional)
- Service icons (SVG format recommended)

### Placeholder Images
Currently using SVG placeholders. Replace with actual images when available.

## Logo Guidelines

- Maintain clear space equal to height of "P" ÷ 3.236
- Primary logo for desktop views
- Secondary logo for mobile views and favicon
- Ensure high-resolution versions are available (SVG preferred)
